An analogy to describe the differences between HTML and CSS

First things first, let’s talk about HTML and CSS.

HTML describes the structure of a Web Page therefore, it consists of inputting elements that will tell the browser how to display them.
On the other hand, CSS works as a styling tool that can be applied to each of the HTML elements. CSS can alter font-names, colours, background -colors , positions and several other properties to make our website stand out.


My personally analogy will be:

Let’s say that you just rented a house and you will need furniture for it. If you use HTML you will be having the basic furniture (elements) to be able to cook, rest and chill on your couch. These are the basic elements to have a functional house. Elseways, if you use CSS, you are able to enhance your house further. This is like acquiring plants, framed art, ect in order to “decorate” the house. You could even pick a certain couch colour so it will match your lounge wall colour. These properties are only decorative and have no real function as HTML does.


Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'. (But not those ones).

Let’s define them first:

Control Flows is in computer science the order that the instructions, statements or functions are executed while some condition.
Looping facilitates the execution of a set of instructions, statements or functions repeatedly while some condition.

This is an analogy of an everyday life activity to understand this concepts.

Let’s say that our control flow function is:

  • If sunny, make a laundry washing
  • Else if (cloudy), check the weather forecast
  • Else (rainy) hold on to it.

This means that if the day is sunny you will do the laundry. If cloudy, you should check the rain forecast for the day.
Otherwise, if it rains, there is no point in doing your laundry.

Now, let’s take this example to a loop function.
On the loop function, you repeatedly execute the instructions(this means, make several laundry washings throughout the day) until you reach one of the ‘else’ statements. On the loop function, you should have a starting point, and an exit criteria so that it won’t loop indefinitely. In this example it will be until it is too late in the day to keep doing the laundry.


Describe what the DOM is and an example of how you might interact with it.

DOM stands for Document Object Model. This is the way that Javascipt interacts with our HTML file.

The way that DOM can be used is by manipulating HTML element's, this means creating, deleting, modifying among several other properties.
A simple example of this will be when you hover over a menu item on a website, and it displays a long list of items within it altering the background colour of the element where our mouse is hovering.

This is a great property to be used when there is a connection between the values.
